Posicionamento Grid Content

Dentro da Grid Area

É raro fixar o tamanho da área do grid de maneira estática (nesse caso em px), pois podemos terminar com um grid pequeno. A maneira mais comum é usando tamanhos relativos (ex: fr).

Grid com tamanho estático: 3 linhas 20px cada ; 3 colunas 20px cada

Geralmente o Grid fica menor do que a área definida quando fixamos seu tamanho estaticamente. Desse modo podemos usar duas propriedades para alinhamento da própria grid:

  • justify-content
  • align-content

Para essas propriedades temos os seguintes valores:

  • start
  • end
  • center
  • stretch
  • space-between
  • space-around
  • space-evenly

Veja os exemplos abaixo e brinque com as combinações que podem ser feitas:

justify-content: center
align-content: center

justify-content: space-between
align-content: space-between

justify-content: flex-end
align-content: align-start
